1/ From an exec statement, remove null parameters. `EXEC @sp @var1=null, @var2=value` becomes `EXEC @sp @var2=value`

2/ Profile SELECT @p1,@p2 becomes param names

3/ Gen debug select based on DECLARE @var1, @var2

4/ Convert select statement to a string. `SELECT @var FROM table` becomes 'SELECT ' + CAST(@var AS NVARCHAR(80)) + ' FROM table'
